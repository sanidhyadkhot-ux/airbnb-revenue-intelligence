# Dataset Source

This repository includes a compressed sample dataset for GitHub and Streamlit demo use.

The full dataset source used for the project is the Airbnb dataset provided by the user, based on the SQL project reference article. The full raw CSV is intentionally excluded to keep the repository under 25 MB and GitHub-friendly.

To reproduce with the full dataset, place the original CSV at:

```text
data/raw/airbnb_wa.csv
```
